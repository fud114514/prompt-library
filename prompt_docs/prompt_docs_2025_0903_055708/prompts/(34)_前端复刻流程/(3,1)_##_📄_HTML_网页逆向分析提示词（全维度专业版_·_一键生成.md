### 2️⃣ 页面结构层级（DOM Hierarchy）

* 页面主结构（Header、Main、Sidebar、Footer 等）

### 3️⃣ 关键 UI 组件列表（UI Components）

### 5️⃣ 色彩系统（Color Palette）

### 6️⃣ 字体与排版（Typography）

* 极简主义（Minimalism）
* Neumorphism（新拟态）
* Material Design

* 少年感 / 游戏化 / 活力感

* 弹窗、下拉、模态框等交互行为
* 表单验证、按钮反馈等用户体验设计

### 9️⃣ 响应式与适配性（Responsiveness & A11y）

* 可否重构为组件化结构（如 React/Vue/Tailwind 组件）
